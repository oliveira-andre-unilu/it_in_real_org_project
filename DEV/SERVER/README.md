# Backend Startup Guide

## Usage

### 1. Put the script in a desired path
Save the shell script as `start_backend.sh`.

### 2. Make it Executable

```bash
chmod +x start_backend.sh
```

### 3. Run the Script
```bash
./start_backend.sh
```

### 4. Reattach to the Screen Session Later
```bash
screen -r timelink_backend
```
